# Bree's Bakery

Angular storefront for Bree's Bakery with cart and WhatsApp ordering.

## Run locally

```bash
npm install
npm start
```

Open http://localhost:4200/

## Build

```bash
npm run build
```

## GitHub Pages

Push to `main` to deploy via `.github/workflows/deploy-pages.yml`.

Live site: https://emmanuel-ratemo.github.io/bakery/

Update the business WhatsApp number in `src/app/data/products.ts` (`WHATSAPP_NUMBER`).
